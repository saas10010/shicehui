# 小程序教师端｜学生看板：强化“时间轴”与“薄弱点”（2026-01-18）

## 目标
- 让“时间轴 / 错题 / 薄弱点”三者差异更明显：
  - 时间轴强调过程与阶段（按日期聚合 + 摘要）
  - 薄弱点强调诊断与优先级（补充口径与样本）

## 改动文件
- `components/mini/teacher/data-dashboard-panel.tsx`

## 时间轴（强化点）
- 新增按日期聚合：将 `wrongByStudent` 按 `createdAt` 解析出的日期分组。
- 每组提供摘要：错题条数、涉及批次数、主要知识点 Top2。
- 支持展开/收起：默认展示每组前 3 条，避免页面“像错题列表一样”平铺。
- 切换学生后重置展开状态，避免跨学生沿用造成理解偏差。

## 薄弱点（强化点）
- 从“仅按知识点计数”升级为多指标：
  - 错误次数、平均错误率（Tag 展示）
  - 最近一次出现日期
  - 涉及批次数
  - 样本题目（最多 3 条）
- 保留原型口径提示：薄弱点基于错题聚合，不等同于掌握度模型。

## 约束
- 仍然只使用前端 mock 数据：三者都基于 `getWrongQuestionsByStudent(studentId)`。
- 当前环境缺少 eslint/tsc，无法执行自动化校验命令；改动已按现有代码风格做静态自检。
